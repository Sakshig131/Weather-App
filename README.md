# 🌦️ Weather App

A simple weather application that shows real-time weather data using OpenWeather API.

## 🚀 Features
- Search city weather
- Shows temperature, humidity, wind speed
- Real-time API data

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## ▶️ How to Run
1. Open index.html
2. Enter city name
3. Get weather details
